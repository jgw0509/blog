[Django 블로그 플랫폼 주요 기능 정리]

1. 게시글 관리 (Post Management)
   - 게시글 목록 조회 (10개씩 페이징 처리)
   - 게시글 상세 조회 (본문, 조회수, 작성일, 댓글 목록)
   - 게시글 작성/수정/삭제 (이미지 썸네일 지원, 마크다운 대응)
   - 발행/비공개 상태 설정

2. 카테고리 기능 (Categorization)
   - 게시글 카테고리 분류 및 필터링
   - 카테고리별 게시물 개수 표시 (사이드바)
   - 신규 카테고리 생성 기능

3. 검색 및 정렬 (Search & Filter)
   - 제목 및 본문 키워드 검색
   - 정렬 옵션 지원 (최신순, 좋아요순)
   - 카테고리별 필터링 연동

4. 사용자 상호작용 (Engagement)
   - 좋아요/싫어요 기능 (게시글 및 댓글 각각 지원)
   - 조회수(Views) 실시간 카운팅
   - 추천 게시물: 상세 페이지 하단에 같은 카테고리의 관련 글 표시
   - 인기 게시물: 조회수 기준 인기 글 사이드바 노출

5. 댓글 시스템 (Comment System)
   - 댓글 작성 및 삭제 (작성자 전용)
   - 대댓글(답글) 기능 지원 (계층 구조)
   - 댓글별 좋아요/싫어요 토글

6. 회원 관리 (User Accounts)
   - 회원가입 (가입 시 자동 로그인)
   - 로그인 / 로그아웃 (Django 5.0 보안 표준 준수)
   - 사용자 프로필 페이지 (회원 정보 및 본인이 쓴 글 목록 확인)
   - 프로필 정보 수정기능

7. 인프라 및 보안 (System)
   - Docker 환경 구축 (web, db 컨테이너 구성)
   - PostgreSQL 기반 데이터베이스 연동
   - URL 슬러그(Slug) 지원 (한글 지원 및 유니크 주소 생성)
